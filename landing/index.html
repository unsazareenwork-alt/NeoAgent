Capabilities

Everything you need to run
your own AI agent

One server. Full-stack agent capabilities. No vendor lock-in.

Android Control

Let the AI control a server-attached Android emulator or device. Screenshots, UI dumps, taps, typing, intents, and ADB shell.

Browser Automation

Navigate, click, type, extract, screenshot, and evaluate JavaScript on any page. Remote control through a Chrome extension.

Recordings & Transcripts

Server-side audio sessions with chunked uploads, Deepgram transcription, segment search, and AI-generated insights.

15+ Messaging Channels

Telegram, Discord, WhatsApp, Slack, Teams, Signal, iMessage, Matrix, IRC, Twitch, LINE, Google Chat, and Telnyx Voice.

Health Connect

Sync Android Health Connect data — steps, heart rate, sleep, exercise, and weight. AI reads summaries through dedicated tools.

Long-Term Memory

Semantic memory, session search, daily logs, and core memory. The agent remembers context across conversations.

Getting Started

Up and running in minutes

Two commands. One config. You're live.

01

Install

Install globally with npm. Works on macOS and Linux with launchd and systemd integration.

~$ npm install -g neoagent
02

Configure

Interactive setup for AI providers, messaging channels, OAuth integrations, and runtime preferences.

~$ neoagent install
03

Deploy

Access the operator UI from any browser. Connect the Android app. Start automating.

~$ neoagent status
● running on :3456
